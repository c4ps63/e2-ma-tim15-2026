# e2-ma-tim15-2026

Projektni zadatak iz predmeta Mobilne aplikacije (2025/26).

Mobilna aplikacija po ugledu na kviz Slagalica — takmičenje igrača kroz više igara (Ko zna zna, Spojnice, Asocijacije, Skočko, Korak po korak, Moj broj).

## Tehnologije

- Java + XML
- Android Studio
- Firebase (Firestore, Authentication, Realtime Database, FCM)

## Arhitektura

Troslojna arhitektura:

- `model/` — klase podataka
- `repository/` — pristup Firebase bazi
- `service/` — poslovna logika
- `activities/` — UI aktivnosti
- `activities/fragments/` — fragmenti
